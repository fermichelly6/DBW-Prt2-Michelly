# Projeto AV1 - Países

Projeto interativo que mostra informações sobre países utilizando API pública.

## Tecnologias
- HTML
- CSS
- Bootstrap
- JavaScript
- API REST Countries

## Funcionalidades
- Lista de 20 países
- Exibição de bandeiras
- Capital, moeda e população
- Consumo de API com fetch
- Modal interativo ao clicar no país
- Informações detalhadas dos países
- Efeitos visuais e animações
- Interface responsiva

## Objetivo
Desenvolver um site informativo e interativo utilizando consumo de API e manipulação do DOM com JavaScript.
